Mocking Life

This mocking workshop we'll have a new take on the code retreat workshop's old friend, Conway's Game of Life. But instead of just writing a rules engine, we're going to implement a physical board, with LEDs that light up to show if a cell is alive or dead.

Because our workstations don't have the same hardware that our fancy art installation Life board does, we need to learn some new techniques to make our code work with this board. In this workshop we'll learn how to test our logic against hardware that we don't have.
